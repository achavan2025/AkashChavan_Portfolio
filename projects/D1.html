Election Dashboard

  • Tech Stack: Python, Advance Excel, Microsoft Power bi
  • Project: Dashboard

The Election Analysis Dashboard provides an interactive overview of election data, featuring filters for District Name, Taluka Name, and Town Name, along with summary metrics like Total Towns, Total Booths, and Total Voters. It visualizes gender distribution through a donut chart showing the percentage and count of male and female voters. Additionally, the dashboard includes bar charts that display the count of voters by Taluka and the count of booths by district and town. A line chart presents the relationship between voter age and the number of booths, while a detailed table offers voter-specific information, such as Town ID, Voter Name, Age, Gender, and Booth Details, providing a comprehensive tool for election data analysis.
